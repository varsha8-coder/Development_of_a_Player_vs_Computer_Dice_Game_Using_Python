# Development_of_a_Player_vs_Computer_Dice_Game_Using_Python

This program simulates a simple dice game between a player and the computer using Python. The game is played for a user-defined number of rounds, where both the player and the computer roll a virtual dice in each round.

The program uses the random module to generate random numbers between 1 and 6, representing the outcome of a dice roll. In every round, the player is prompted to press Enter to roll the dice. Both the player’s and the computer’s dice values are displayed, and the winner of the round is determined based on whose value is higher.

The program keeps track of scores for both the player and the computer across all rounds. After all rounds are completed, the final scores are displayed, and the program announces the overall winner. If both scores are equal, the game ends in a tie.
